함수(Function)

선언적 함수

반환 값 선언없이 function 키워드만 이용하여 사용
function 함수명() { 소스코드 작성 } 으로 작성해서 실행

익명 함수

function에 함수명을 작성하지 않고 변수에 대입하는 방법
함수명 = function() { 소스코드 작성 }으로 작성해서 실행

스스로 실행하는 함수(즉시 실행 함수, IIFE(Immediately Invoked Function Expression))

(function() {소스코드 작성})();로 작성

함수의 전달인자와 매개변수

함수의 리턴처리

매개변수로 함수 전달(feat. callback 함수)

익명함수를 매개변수로 활용
함수도 하나의 자료형이기 때문에 매개변수로도 전달 가능함.
