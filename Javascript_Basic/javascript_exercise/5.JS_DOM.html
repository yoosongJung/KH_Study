DOM(Document Object Model)

문서객체 : HTML에 있는 태그를 객체화하여 자바스크립트로 다룰 수 있게 한 것,
모든 노드 객체에 접근할 수 있는 요소와 메소드를 제공.


노드 : HTML에 있는 태그를 구조화(트리화)하였을 때 각각의 태그가 노드임.

요소노드(elements node)와 텍스트노드(textnode) : 요소노드는 태그 자체를 의미하며 텍스트노드는 태그에 기록되어 있는 문자임.

노드의 생성과 삭제

절차 : 요소노드 생성 -> 텍스트노드 생성 -> 요소노드에 텍스트노드 추가 -> body객체에 요소노드 추가

요소노드는 텍스트 노드가 있는 노드와 없는 노드가 존재함.

텍스트 노드가 있는 노드 생성

ex) div, p, span, ...

텍스트노드가 없는 노드 생성

ex) img, input, ...

JS를 이용한 HTML태그 접근

JS를 이용하여 HTML태그를 동적으로 처리하기 위해서는 DOM객체의 메소드(함수, 명령어)를 호출하여 접근해야 함.

메소드(함수, 명령어)

1. getElementById("아이디값") : 태그의 id 속성값을 이용해서 태그 요소 객체 정보를 가져옴(1개)
2. getElementsByName("이름값") : 태그의 name 속성값을 이용해서 태그 요소의 객체 정보를 배열에 담아 가져옴(여러개)
3. getElementsByTagName("태그명") : 태그명을 이용해서 태그 요소의 객체 정보를 배열에 담아 가져옴(여러개)
4. getElementsByClassName("클래스값") : 태그의 class 속성값을 이용해서 태그 요소의 객체 정보를 배열에 담아 가져옴(여러개)
