이벤트

DOM programming 의 경우
document.querySelector("#btn").addEventListener("click", function() {실행할 코드 작성});

jQuery 의 경우
$("btn").on("click", function() {실행할 코드 작성});

이벤트 종류

마우스 관련
click, dblclick, mousemove, mouseover, mousedown, mouseup
키보드 관련
keyup, keydown(한글 감지), keypress(숫자, 영어 감지)
Form 관련
submit, focus, blur, change
